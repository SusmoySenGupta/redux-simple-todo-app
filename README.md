# Redux Simple Todo Application

Simple Todo Application built with React, Redux. It is actually a classwork from the course "Think in a Redux Way" by @LearnWithSumit.

## Features

-   Add Todo
-   Delete Todo
-   Toggle Todo
-   Filter Todo

## Tech Stack

-   React
-   Redux
-   Tailwind CSS
